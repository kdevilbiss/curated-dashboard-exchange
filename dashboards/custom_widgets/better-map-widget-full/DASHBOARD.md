# Better Map Widget (Full)

Enhanced map dashboard with marker clustering, status detail, weather overlays, and resource or group mapping options.

## Attribution

- Original creator: Kevin Ford
- Curator/packager: Kerry DeVilbiss
- Permission/status: public-upstream
- Original source: https://github.com/logicmonitor/custom_widgets/

## Requirements

- LogicMonitor Text widget raw HTML mode.
- Valid location properties on mapped groups, resources, or services.
- External map/CDN access from the viewer browser.
- Optional weather API keys for weather overlays.

## Known Limits

- Custom-developed widget; standard LogicMonitor Support may be limited.
- Weather and overlay features depend on external API availability.
- Contains raw HTML/JavaScript Text widget code.
