# Attribution

Curated Dashboard Exchange is attribution-first. The goal is to make useful dashboards easier to find and evaluate while preserving authorship and context.

## Inspiration

This catalog was inspired in part by Kevin Ford's Better Map widget work and by the broader practice of using raw HTML Text widgets to extend LogicMonitor dashboard experiences.

## Curator

Kerry DeVilbiss is the initial curator and packager for this repository.

## Per-Dashboard Attribution

Each dashboard package must record:

- Original creator.
- Curator/packager.
- Permission or publication status.
- Original source when known.
- Material changes made during curation.
- Requirements, dependencies, and known limitations.

Do not publish a dashboard package until the attribution and permission/status fields are filled in.
